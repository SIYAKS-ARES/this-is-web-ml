# Real-time Object Detection — COCO-SSD (TF.js)

- Web kameradan görüntü alır.
- COCO-SSD ile nesne tespiti yapar ve etiketli kutular çizer.

## Çalıştırma

```bash
python3 -m http.server 8000
# veya
npx http-server -p 8000 --silent
```

Tarayıcıda açın ve kamera izni verin:

- http://localhost:8000/examples/real-time-object-detection/
