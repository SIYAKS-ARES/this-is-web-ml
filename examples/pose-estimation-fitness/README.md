# Pose Estimation Fitness — MoveNet (TF.js)

- Web kameradan görüntü alır.
- MoveNet ile iskelet noktalarını çizer ve basit squat sayacı tutar.

## Çalıştırma

```bash
python3 -m http.server 8000
# veya
npx http-server -p 8000 --silent
```

Tarayıcıda açın ve kamera izni verin:

- http://localhost:8000/examples/pose-estimation-fitness/
