# Image Classification — MobileNet (TF.js)

- Kullanıcı bir görsel yükler.
- MobileNet ile sınıflandırma yapılır ve en iyi 3 tahmin gösterilir.

## Çalıştırma

```bash
python3 -m http.server 8000
# veya
npx http-server -p 8000 --silent
```

Tarayıcıda açın:

- http://localhost:8000/examples/image-classification/
