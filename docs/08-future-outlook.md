## 08 — Future Outlook

### WebNN API

Tarayıcıda nöral ağ çıkarımı için yüksek seviyeli ve donanım-agnostik bir API hedefleniyor. Tarayıcı, yerel hızlandırma katmanlarına (DirectML, Core ML, Android NNAPI) köprü görevi görecek.

### In-Browser Generative AI

WebGPU ile daha büyük ve üretici (generative) modellerin tarayıcıda çalışması mümkün hale geliyor (ör. Stable Diffusion türevleri, hafif LLM’ler). Model küçültme/quantization teknikleriyle kapsam genişliyor.

### Deeper Web Platform Integration

WebXR, WebRTC, Web Audio gibi standartlarla bütünleşik, bağlamsal ve gerçek zamanlı deneyimler kaçınılmaz görünüyor.
