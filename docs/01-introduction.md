## 01 — Introduction to Web Machine Learning (Web ML)

### What is Web ML?

Web Machine Learning (Web ML), makine öğrenmesi çıkarımını doğrudan web tarayıcısında çalıştırmayı hedefleyen bir yaklaşımdır. Amaç; gizliliği koruyan, düşük gecikmeli, maliyeti düşük ve çevrimdışı çalışabilen zekâ özelliklerini standart web teknolojileriyle hayata geçirmektir.

### Mission and Rationale

- ML’yi birinci sınıf web vatandaşı yapmak
- Modern tarayıcıların GPU/CPU yeteneklerini kullanarak gerçek zamanlı kullanıcı deneyimleri sunmak
- Veriyi cihazda tutarak güvenliği ve gizliliği artırmak

W3C Web Machine Learning Community Group (2018) ve devamındaki Working Group çalışmaları; tarayıcıda çıkarımı kolaylaştıran API’lerin olgunlaşmasına odaklanır.

### Disambiguation: Web ML vs. WebML (Web Modeling Language)

- **Web ML (bu rehberin konusu)**: Tarayıcıda ML çıkarımı (WebGL, WebGPU, Wasm gibi teknolojilerle hızlandırma).
- **WebML (Web Modeling Language)**: 2000’ler başından bir kavramsal modelleme dili ve yöntemidir; makine öğrenmesi ile ilgili değildir ve web sitelerinin yapısal/hypertext/presentational/personalization modellerini tanımlar. Güncel Web ML ile karıştırılmamalıdır.
